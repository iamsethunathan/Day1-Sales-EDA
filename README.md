# 📊 Superstore Sales Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a Superstore Sales dataset to uncover business insights and sales trends.

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📊 Key Analysis Performed
- Total Sales & Profit Calculation
- Top 10 Selling Products
- Category-wise Sales Analysis
- Region-wise Profit Analysis
- Monthly Sales Trend Visualization

## 📈 Key Insights
- Technology category generated the highest revenue.
- Sales peak observed during festive months.
- West region contributed highest profit.
- Some products have high sales but low profit margin.

## 📁 Dataset
Superstore Sales Dataset (Kaggle)

## 🚀 Conclusion
This project demonstrates foundational data cleaning, visualization, and business insight extraction skills.
